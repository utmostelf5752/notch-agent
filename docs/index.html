Hover the notch — the panel drops out of it.

Small surface, real agent

Opens without stealing focus

Spotlight-style non-activating panel: it takes your keystrokes while the app you're in stays frontmost. Esc or click away to dismiss; pin it to keep it up while you work.

Your CLIs, headless

Messages run through the claude, codex, or Cursor agent CLI you're already signed into — your accounts, your usage, your working folder.

ChatGPT without an API key

A hidden web view logged into chatgpt.com sends prompts and scrapes replies. Sign in once; threads resume across restarts.

Streaming, with the work shown

Replies stream in token by token, and tool activity renders live, then folds into an expandable "N steps" row when the turn finishes.

Drop files on the notch

Drag files onto the panel or the notch itself, or grab a screenshot of the full screen or the active window from the paperclip menu.

Chats that come back

Every conversation is archived with its provider and session id. Restore one and it continues where it left off — including ChatGPT threads.

Install

01

Download the DMG and drag NotchAgent to Applications.

02

First launch: the app is ad-hoc signed, so Gatekeeper blocks a plain double-click. Right-click → Open → confirm, or clear the quarantine flag:

xattr -dr com.apple.quarantine /Applications/NotchAgent.app
03

Hover the notch (or press ⌥ space), pick a provider and a working folder in the composer, and say hi.
